A high-performance, immersive developer portfolio built with **Next.js 14**, **TypeScript**, and **Tailwind CSS**. Designed with a focus on "Deep Black" aesthetics, fluid WebGL animations, and a seamless user experience.

- **Immersive 3D Hero:** Custom WebGL fluid simulation shader.

- **Seamless Transitions:** Smooth gradient fades between sections for a continuous environment.
- **Interactive Navigation:** Smart `CircleMenu` that stays out of the way and auto-closes.
- **Performance Optimized:** Clean code structure, efficient react-three-fiber usage, and optimized assets.

## üì¶ Getting Started

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/Yash13606/Portfolio_Website.git
    cd Portfolio_Website
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```
